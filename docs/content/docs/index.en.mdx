## Project Overview

This project implements an ETL process to integrate data from multiple transactional sources into a centralized SQL Server Data Warehouse.

### Supported Databases

- **Microsoft SQL Server**: Transactional sales data in USD
- **MySQL**: Transactional sales with alternate codes and text-formatted amounts
- **MongoDB**: Document-based sales with totals in CRC
- **Neo4j**: Graph database for purchase relationships
- **PostgreSQL/Supabase**: UUID-based transactional system

## Quick Start

Once all dependencies are installed, you can quickly start all databases:

**Linux/macOS:**
```bash
./scripts/dev_up.sh
```

**Windows:**
```powershell
.\scripts\dev_up.ps1
```

For more details on using these scripts, see the [Database Setup](/en/docs/installation/database-setup) guide.
